ヴォルカバンバ (Volcabamba)
2010/08/21 ver 1.0
2010/09/03 ver 1.1 2ボタンモード、キーコンフィグの追加

Windows 98 以降対応


○インストール
    volca.zip を解凍して適当な所へ移動させてください

○アンインストール
    ゲームが入ってるフォルダを削除してください
    レジストリはいじってません

○起動方法
    volca.exe をダブルクリックで起動します

○終了方法
    ESCキーで終了します

○ゲームの始め方
    タイトル画面でスペースキーを押すとゲームが始まります

○ゲーム中の操作
    カーソルキーで8方向に移動
    ジョイスティックにも対応しています

    ショットは2方向に発射されます
    ゲーム開始時にショット方向変更方法を以下の二つから選べます

    1)ショットボタンを離してる時に移動した方向に変わる
    2)ショット方向切り替えボタンを押すたびに変わる

    ※ 2)を選択した時のみ、8方向モードか4方向モードを選べます
       慣れないうちは、2)の4方向モードがオススメです

    キーボードでの操作 
    ショット : スペース, Z
    切り替え : X, C, V, B, N, M
    ※ 一部のキーはキーコンフィグで変更できます

    F1キー を押すとポーズです

○ゲーム内容
    ボスを倒すとステージクリアのオーソドックスなシューティングです
    全6面

    最初は2万点で残機が増え、それ以降は5万点ごとに増えます

    エリア1から5までの間には「ヴォルカ石球」が隠されています
    隠されている場所にショットを撃ち込むと出現します
    出現させると100点、取ると1000点です
    取ったヴォルカ石球は画面下に表示されます

○コンフィグ
    v_config.exe をダブルクリックすると起動します

    キーコンフィグ、フルスクリーンモードかウィンドウモードかの選択が出来ます
    SHOT   : ショット	
    CHANGE : ショット方向切り替え
    PAUSE  : ポーズ

○なんでわざわざMSX風にするの？
    作者はMSX風に絵を描いたり動かしたりする事に興奮する末期患者なのです
    近寄っちゃダメ

○ゲーム作成環境
    このゲームはオニオンソフトさんの、HSP3.2 (Hot Soup Processor)という言語で
    作成されています
    http://hsp.tv/ 

    ジョイスティック入力には
    MIAさん作成の (mod_joystick.as) を使用しています
    http://taillove.jp/mia/

    HSP時間管理の精度アップには
    みなりりさん作成のTime Manager for HSP3 (tmanage3.as) を使用しています
    http://sakmis.6.ql.bz/

    サウンド再生には (hspogg.dll) を使用しています
    -------------------------------------------------------------------
    OggVorvis形式のファイルデコードに以下のライブラリを使用しています
    libvorbis : Copyright (c) 2002-2004 Xiph.org Foundation
    -------------------------------------------------------------------
    Oggファイルは使って無いんですけどね
    なぜかウチの環境ではHSP上で Ogg と MP3 が再生されない
